# 🌿 Фитотека

**Умный справочник лекарственных трав**

---

## 📱 О приложении

Мобильное приложение для подбора лекарственных растений по симптомам. Содержит подробную информацию о 18+ травах, их свойствах, способах приготовления и противопоказаниях.

---

## ✨ Ключевые функции

| Функция | Описание |
|---------|----------|
| 🔍 **Подбор по симптомам** | Выберите симптомы, получите список подходящих трав с процентом совпадения |
| 📚 **Энциклопедия** | Полная информация о каждом растении: описание, польза, применение, приготовление |
| ⭐ **Избранное** | Сохраняйте любимые травы |
| 💡 **Советы и факты дня** | Ежедневные полезные рекомендации |
| 📸 **Реальные изображения** | Качественные фото каждого растения |
| 📊 **Рейтинг и отзывы** | Оценка каждой травы на основе отзывов |

---

## 🌱 База данных

| Параметр | Значение |
|----------|----------|
| Количество растений | **18+** |
| Симптомов для подбора | **28+** |
| Категории | Успокаивающие, Иммунитет, Пищеварение, Кожа, Сердце |

---

## 🛠️ Технологии

- **React Native** — кроссплатформенная разработка
- **Expo SDK 54** — инструмент сборки и тестирования
- **React Navigation** — навигация между экранами
- **expo-linear-gradient** — градиенты и современный дизайн
- **react-native-animatable** — плавные анимации

---

## 🚀 Запуск проекта

### Требования
- Node.js 18+
- npm 9+
- Expo Go на телефоне

### Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/Anna-Shuyanova/herbal-guide.git
cd herbal-guide

# Установка зависимостей
npm install

# Запуск приложения
npx expo start
